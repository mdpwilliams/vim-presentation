Learning Broken Down

  1. New Thing
  2. "The Dip"
  3. Appreciation and Growth
  • At first learning something, it's a new thing, interest in this is holding you to it.
  • Then, things start grinding and feeling icky, bad. It is difficult and you finally start realizing the goal is further than you had anticipated. Pushing past this dip is the key part of learning.
  • Start achieving results and it clicks, and you can make incremental gains towards success and mastery.

Installation

Windows

Install it from www.vim.org specifically the self-install executable

Specifically, the self-installing executable.

Linux/Mac

Tends to come with vim already

If not, you can install with 

apt-get install vim
on Linux and 
brew install vim
on Mac.

In addition, there are a number of hotkey extensions and plugins for an assortment of different editors to create vim-like functionality.

  • VSCode's Vim
  • Sublime Text 3's Vintage Mode
  • Atom's vim-mode-plus
  • PhpStorm's IdeaVim

Please note that not all functionality passes over.

I learned vim at first using Atom and vim-mode-plus!

Opening/Closing Files

If you are using a plugin for your editor of choice, stick with their method of opening files.

To open a file, using the terminal you need a command like:

vim path/to/file.txt
